# Hugo


**Connecting dog owners with trusted walkers.**

Hugo is a mobile app that connects busy dog owners with verified walkers, so dogs get the regular exercise they need even when their owners can't be there. It also helps owners learn how to take better care of their pets through short, story-style training and care tips.

Project presentation: [Hugo](https://canva.link/i777hwi372nxtit)
 
---

## The problem

- Many dog owners struggle to find time for regular walks.
- Lack of proper exercise can lead to behavioral and health issues in dogs.
- Finding a trustworthy dog walker is often challenging.

## The solution

A mobile app that:

- Connects dog owners with verified walkers
- Ensures dogs get regular exercise even when owners are busy
- Provides training tips, pet care info, and an in-app chat for direct communication

## Getting started

### Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/Ghostx-x/Hugo.git
   cd hugo
   ```

2. Open the project in Android Studio (`File > Open` → select the cloned folder).
3. Let Gradle sync.
4. Run the app on an emulator or a connected device.

### Running on a physical device

1. Enable Developer Options (tap **Build number** seven times in **Settings → About phone**).
2. Turn on **USB debugging**.
3. Connect the phone via USB and pick it as the run target in Android Studio.
